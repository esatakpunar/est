~9KB total

7.1KB CSS + 2.4KB JS, minified and gzipped. Nothing more, nothing less.

Zero dependencies

No Node.js ecosystem. No React, Vue, or any framework. Standalone CSS and vanilla JS.

Semantic HTML

Native <button>, <dialog>, <details> styled directly. No class soup.

Accessible

ARIA roles, keyboard navigation, focus management built in throughout every component.

Dark mode

Automatic dark mode via prefers-color-scheme. All tokens adapt. Override with one attribute.

Customizable

Override CSS custom properties to match your brand. No rebuilding, no config files.

Why est?

Built from oat with a different visual language — soft purple accent, Apple HIG-inspired spacing, Notion-style underline tabs, and cubic-bezier(0.25, 0, 0, 1) easing everywhere. Same semantic HTML philosophy, same zero-dependency architecture. Different aesthetic.

The goal: a minimal component library that still looks considered. CSS variables for everything, so customization is one line. No build step, no ecosystem lock-in. Include two files and ship.
